    关于工作

    VS Code 在 每个夜晚看着我

    从15年接触前端实习,到16年正式开始了一段难得的前端生涯

    面试的时候 seth 温和谦逊, 俊哥 不苟言笑, 后面接触下来 都是可爱的人. 面试过程堪比电影桥段,戏剧性十足, 还是要感谢娜姐 : )

    从最开始的 熟悉 BackBone ,Git, shell 命令 , 无聊的 修改 页面,改改bug

    到后来 写一些 nodejs 的 简单接口

    再到后来的 React, Webpack 和 mongodb, Graphql

    记忆犹新的是 每次无法解决的bug,各种奇葩的问题,都在俊哥耐心的帮助下解决了,

    规范的 敏捷开发, code review git 提交规范, 代码规范,

    很厉害 很好相处的 赵国平,超哥,申鑫,俊哥,seth,杨军, 很厉害的测试 干尚奇 等等

    那一段的时间 是 最累的 也是学到最多的, 几乎每周一次的 技术分享,让我学到很多的装逼技巧,也让我 认识到 什么才是真正的前端,以及 自己和他们的差距，佩服他们做事情写代码的那种认真劲头,他们才是真正的程序员.. 而我只是码农.可惜因为 拖欠工资 大家都走得差不多了

    难得的一次工作经历.感觉以后很难遇到这么好的技术氛围的公司了

    2017 也是我敲 代码最多的一年

    学的好累啊

    这几年,前端变化实在是TM的太快了, 16年累积的那点jquery技术已经跟不上时代了,三天两头出来一个新东西,怎么办呢? 只能咬紧牙关努力敲代码啊!

    • webpack 4 都快出来了, 从webpack1 看着慕课网 学习配置 都要崩溃的我, 到现在能自己手动配置,感觉还是不错

    • 习惯 用yarn 代替 npm , 缓存功能很不错

    • nodejs 8 稳定运行, 支持 async await 写起来不要太爽, 从最开始只会 express 框架, 到 跟着 饿了么开源教程 系统的学习了node 的基础 很底层的一些细节原理,感觉装逼更加有资本了

    • react 发布了 16, vue2 风头正劲, 服务端渲染也愈发火热, 独爱 react 哈哈

    • 刚接触redux的时候 真的是 无处下手 概念难懂, 不过熟悉把流程搞懂之后 发现 就那么回事, 现在又觉得 redux繁琐 mbox 又开始流行起来, 真的恶心,一天一个库 让不让人活了 :(

    • es6,7 到年末 已成为每个前端必会的东西,确实用起来很爽,当初为了搞懂也下了点功夫,现在,哈哈,so easy. 目前 装饰器还有点打脑壳

    • 函数式编程,typescript 越来越多的人关注, PWA 渐进式 web app, 我都学习并尝试过.目前只属于略懂皮毛

    • 最初的 less,sass, 到现在 的 –var(xx) 原生支持更多更多的特性,感觉整个生态再向一个好的方向发展

    • 喜欢上逛知乎,看阮一峰, 张鑫旭 等大牛 的博客 微博 学习 最新的东西, 前者感觉文章越来越水了…

    • 开始喜欢造一些垃圾轮子,学会了怎么在npm上发布包, 就算有一个star 也会很开心

    • css in js 是真的恐怖,不适合我

    • 开始有强迫症了, 代码后面有分号 就头皮发麻

    • 疯狂的学习 让我有点浮躁,和疲劳。 过多的技术 都只懂皮毛, 开始要求自己 知其所以然

    希望 2018 自己 在前端 能更上一层楼，蚂蚁竞走十年了,鹿小葵你可以的 :)

    关于自己

    我比较慢热,有时候有点害羞,有时候又比较骚,有时矫情,有时逗比

    初中成绩还排的上号，后来就开始厌学了,越来越重的抵触心理,讨厌那该死的应试教育,高三上册觉得自己这成绩,天天上课睡觉,玩 没什么意义, 没怎么考虑就背着书包,就休学了. 忘不了回到家 妈妈 看我背书包回来,第一句话就是 你不读啦？ 哈哈 真的是亲妈,后面给父母说想去学 计算机, 后面去读了个 电子科技大学 成人大专..自己也算名牌大学生了？exo !

    现在想想，还是不后悔自己当初的决定, 没有这个转折点 我现在也干不了web前端这一行,以前的努力 ,现在还是有回报,也算对得起父母和自己了!
