    • 2019 年终总结

    全面小康的时代来临啦

    一不留神，2019 年就这样过去啦，看着 2018 的年终总结，仿佛就是前几天写的 😁, 写这段文字的时候，已经是 2020 年 1 月 4 号了，一个号称全面小康的时代，小时候的未来世界，现在看来，除了成都很有未来感的雾霾之外，其实也没什么差别

    去年的目标完成了吗？

    2018 年的目标

    • 学好 Flutter
    • 维护 cuke-ui
    • 巩固 node.js
    • 学习 后端相关知识
    • 学习 Docker
    • 认识更多好玩的人

    好的好的，假装自己全部做到啦 :)

    今年都干了些啥

    这一年的前端圈，似乎步子慢了一些，稳扎稳打的在越变越好。去年我过于浮躁，所有新东西都去沾一点，懂点皮毛，反而把自己搞的很累，今年我给自己做了一定的 ‘减法’, 主要加强了一下自己的基础知识，不再刻意去追求最新最炫的技术。

    工作

    • 参与开发了大半年的 星巴克预约平台, 收获颇多的项目，一看到这些页面，就能想到平时开发中的点点滴滴，code review 的争吵，改 bug 的痛苦，以及完成复杂功能的喜悦

    • 做了一个知名银行的管理系统，如愿以偿实战了 Graphql 和 nest.js, 体验了 BFF, 总的来说，过了一把在公司用 Node.js 的瘾，很爽，学习到很多后端的思维

    • 最后是人生的第一次出差，去了北京，见了见世面，免费旅游了一波 :), 北京的雾霾竟然没有成都严重，不过北京吃的确实比不上成都，人也不行，马路上乱糟糟的，行人乱穿马路，人和车互不相让。比谁的胆子大，另外，风是真的大！贼冷。

    赶上了北京的一场大雪，南方人表示巴适得板

    最后去了下我的母校哈哈

    • 18 年面试了蚂蚁，今年再战，还是失败告终，自己还是继续加油吧

    社区

    吹了好几年的 PWA, 似乎也成了大家口中的笑谈，现在大佬们都在吹啥呢？是 serverless 的火爆，写一堆 云函数，大厂们的小程序也支持云函数，改变开发的生态？还是今年大火，被更多人接受的 TypeScript? 还是比较小众，但势头正猛的 nest.js? 工作上的原因，前端三大框架都用过，还是对 React 情有独钟，不知道 React Hooks 有没有真香的感觉呢？我的感受是前端今年一直在变，却又似乎什么也没变，不比追逐潮流，只用最适合自己的。

    开源项目

    给 Antd 提了一个 PR, 现在也可以吹自己是贡献者了

    自己写的 React UI 库，虽然没维护了，还是得到了 388 个 star, 很满足

    自己随便写的一个 早期用于自己个人网站的播放器，没想到还是有很多国外的开发者在使用，修复了很多 bug, 2 年来，也新增了一些功能尽管很简单，自己一直坚持着维护，还是有些许感叹

    最后就是今年准备写的一个 React Hook 库，名字叫 react-7h-hooks , 平时做业务会有一些重复的工作，都把它们封装成 hook 来复用，让每天少上一个小时的班，希望自己能够坚持下去

    书籍

    我自己很讨厌世面上的 <<深入 xx 框架>>, <<30 天学会 xx>>, 还是需要阅读一些提升自己眼界，基础的书籍

    不过不知道是不是我的原因，很火的 重构 和 代码整洁之道, 我觉得一般，自己看不进去，后者只看了一半。..

    最后悔的 狼书, 前端网红的一本 Node.js 的书籍，通篇夸夸其谈，全讲些概念和虚的，除了那章源码分析，没啥干货，前端架构师就这水平？

    最喜欢的事 漫画算法 和 码农翻身, 对我这种数据结构和算法 完全 0 基础的 三流前端 非常友好，算是自己的一个基础入门

    你不知道的 js 三本 包装都还没拆，还没来的及看。哈哈

    虽然自己阅读量很少，但是总有一种感觉，就是现在是个人就能写书，自己水平不咋样的也敢写书，书籍质量水分太大，和这个浮躁的前端圈有关系？

    音乐

    音乐是一个好东西，15 年开始，喜欢听民谣和说唱，宋冬野和谢帝都非常的叼，今年喜欢上了 C-Block, 喜欢他们的歌词，也喜欢听点灯人乐队，一个重庆的乐队，风格清新，旋律好听，喜欢说唱是喜欢那种直白的表达方式，可能也和自己的心境有关系，以前喜欢装文艺，听一些转基因民谣觉得自己特有逼格，现在想想好搞笑哈哈

    还是坚持着弹吉他，不过今年频率有点低了，只学会了一首新歌，押尾桑的 ‘风之诗’, 一首安静美好的歌曲，希望今年能学会 岸部真明的 ‘The End of the world’, 太好听了最后，希望自己在 YouTube 上 自学 歌曲制作，写了一首说唱歌曲，网易云也找好了伴奏，就是不知道怎么混音和录音，希望上半年能完成这个，送给自己一个礼物，skr~

    付费课程

    今年利用业余时间，在实验楼的要求下，做了一个付费的教程，基于 TypeScript 和 React Hook 的组件化开发实战, 也算是一种新的尝试

    未来

    在父母的帮助下，买了一个小公寓，但是还远远不够，希望自己再存两年前，能够一个房子的首付，拥有属于自己在成都的房子，对于生活，我希望自己依然是热爱生活，足够简单的人，一个做着自己喜欢的事情，不管代码也好，音乐也好，对得起自己的青春，19 岁出来上班的时候，觉得自己特年轻，不知不觉，现在都 23 岁了，除了感叹时间过得真快之外，也多提升自己吧。李金珂同学，加油呀！
