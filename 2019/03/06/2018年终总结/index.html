    • 2018 年终总结

    1. 前言

    一转眼 2018 年就这样过去了，搬了一年的砖，感觉元气满满

    2. 工作

    到今年就三年工作经验了，有一种 一年工作经验重复了三次的感觉 : ( , 今年下半年换了工作，跳槽到了 Thoughtworks, 换个一个地方搬砖，见识和 视野 得到了较大的提升，遇到一群和牛逼 和 好玩的人，到年底，做了两个项目 宝格丽 和 星巴克, 很累学到也很多



    3. 学习

    今年参加了 前端社区 FCC 前端大会, 体验了一把 志愿者 , 其实就是 跑腿打杂，给大佬们 递话筒，还没钱 真·志愿者，哈哈，不过 干货还是挺多，死月等大佬介绍了 nodejs c++相关的 只是，Taro 的 作者 秀了一把语速。..



    大佬们的合照 ↓



    公司的技术 还是老样子，还是 React, Vue, 到了大公司 前后端分工比较明确，应该没机会 接触我爱的 node 了

    业余时间 还是玩了玩 nest.js 框架，让我体验了一种 用 sping boot 写 node 的感觉

    • 在项目中学会了以前没有接触过的 typescript, 说实话我以前是有点抵触这种类型的 超集 的，不过 真香！, 在实际开发的过程中 还是比较爽，定义好 interface 后 , 配合 vscode 的智能提示，非常好用
    • 用了下 redux-saga 真·模板代码，感觉比较无聊，没体会到好处

    个人感觉 今年是 技术长进 最慢的一年，没接触到什么新东西，全靠 自己 业务时间 学习

    唯一比较欣慰的是 花了三个月搞了一个 react 的 组件库 cuke-ui, 竟然有 300 star, 巴适得板，主要是有一天突发奇想，说写一个 锻炼下自己，和几个朋友 一起说开发 就搞起来了，妈的 结果 就我一个人写



    4. 生活

    买了一个小米滑板车，天天骑着去上班，再也不用挤地铁 和 抢共享单车了

    吉他 现在偶尔弹一下在网易云 听了一首 无题, 发现哇塞 贼骚气，动作帅，一定要学会，功夫不负有心人，

    去了一次简单生活音乐节



    感受了一次 秋衣男神 朴树 的现场，贼有感觉，直到看见平凡 才是唯一的答案！

    有了一个 小公寓，人生的第一个房子！, 虽然很小，但是也满足了

    和女朋友还是小吵小闹，不知不觉就一年了

    5. 感悟

    扫码感觉是越来越方便，现在坐地铁公交都是手机扫码，便利店竟然安装了扫脸支付的设备，

    不明白为什么那么多人 走路都在玩手机，送外卖的 边开车边 玩手机 , 骑自行车的 也是边骑边玩手机，走路玩，吃饭玩，拉屎玩，不是很理解这些人

    年味 越来越淡了，没有小时候的那种感觉了，难道是因为要给别人红包了 :) ? 哈哈

    6. 未来

    希望自己还是保持对 技术的 那一份 赤子之心，不停的学习，折腾新，好玩，有趣的技术，明年希望

    • 学好 Flutter
    • 维护 cuke-ui
    • 巩固 node.js
    • 学习 后端相关知识
    • 学习 Docker
    • 认识更多好玩的人

    就像我经常听的那首 空帆船 一样，我爱这艰难而又拼接全力的每一天 !!!

    最后也希望家人，亲人，所有认识我的，和我认识的人，都能好好的！!
