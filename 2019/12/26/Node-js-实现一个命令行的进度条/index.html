    标准输入与输出流

    在 Node.js 中，标准的输入输出流分别为：

    • 输入流
    • 输出流
    • 错误流

    而流又分为：

    • 可读流 (Readable)
    • 可写流 (Writable)
    • 双工流 (Duplex)
    • 转换流 (Transform)

    其中双工流和转换流都是可读可写的。
